# ObsidianTUI

ターミナルで動作するObsidian Vault向けのTUIクライアントです。

## 特徴

- **ファイルツリー**: Vaultのファイル構造をツリー表示
- **ライブエディタ**: WYSIWYGスタイルのMarkdownエディタ（TeX数式対応）
- **プレビュー**: Markdownのリアルタイムレンダリング
- **グラフビュー**: ノート間のリンクを可視化
- **バックリンク/フォワードリンク**: リンク関係の表示
- **タグペイン**: タグ一覧とフィルタリング
- **アウトライン**: 見出し一覧とジャンプ機能
- **デイリーノート**: 日付ベースのノート作成
- **コマンドパレット**: 全機能への素早いアクセス
- **埋め込みノート**: `![[note]]`構文のプレビュー展開
- **Wikiリンク**: `[[note]]`リンクのナビゲーション

## インストール

### Go Install

```bash
go install github.com/nyanko3141592/ObsidianTUI@latest
```

### ソースからビルド

```bash
git clone https://github.com/nyanko3141592/ObsidianTUI.git
cd ObsidianTUI
go install .
```

## 使い方

```bash
# Vaultディレクトリで実行
cd /path/to/your/vault
obsidiantui .

# またはパスを指定
obsidiantui /path/to/your/vault

# 引数なしで前回のVaultを開く
obsidiantui
```

## キーバインド

### グローバル

| キー | 機能 |
|------|------|
| `Ctrl+C` / `Ctrl+Q` | 終了 |
| `?` | ヘルプ表示 |
| `F1` | コマンドパレット |
| `Tab` | 次のペインへ移動 |
| `Shift+Tab` | 前のペインへ移動 |
| `Alt+1` | ファイルツリーにフォーカス |
| `Alt+2` | エディタにフォーカス |

### ビューモード

| キー | 機能 |
|------|------|
| `Ctrl+E` | ビューモード切替 |
| `Alt+E` | 編集モード |
| `Alt+P` | プレビューモード |
| `Alt+S` | 分割モード |

### ファイル操作

| キー | 機能 |
|------|------|
| `/` / `Ctrl+P` | ファイル検索 |
| `Ctrl+S` | 保存 |
| `Ctrl+N` | 新規ファイル |
| `Ctrl+D` | 削除 |
| `Ctrl+R` | Vault再読込 |

### ナビゲーション

| キー | 機能 |
|------|------|
| `Ctrl+]` / `gd` | リンク先へジャンプ |
| `Ctrl+O` | 戻る |
| `Ctrl+B` | バックリンク |
| `Alt+F` | フォワードリンク |

### Obsidian機能

| キー | 機能 |
|------|------|
| `Ctrl+G` | グラフビュー |
| `Ctrl+T` | タグペイン |
| `Ctrl+L` | アウトライン |
| `Alt+D` | デイリーノート |

### エディタ/プレビュー

| キー | 機能 |
|------|------|
| `j` / `↓` | 下へ移動 |
| `k` / `↑` | 上へ移動 |
| `g` | 先頭へ |
| `G` | 末尾へ |
| `Ctrl+U` | ページアップ |
| `Ctrl+D` | ページダウン |

### オーバーレイ

| キー | 機能 |
|------|------|
| `Esc` / `q` | 閉じる |
| `Enter` | 選択実行 |

## 対応フォーマット

- Markdown (`.md`)
- Wikiリンク: `[[ノート名]]`, `[[ノート名|表示テキスト]]`
- 埋め込み: `![[ノート名]]`
- タグ: `#tag`, `#nested/tag`
- TeX数式: `$inline$`, `$$block$$`

## 設定

設定ファイルは `~/.config/obsidiantui/config.json` に保存されます。

## 必要要件

- Go 1.21以上
- ターミナルエミュレータ（マウス対応推奨）
